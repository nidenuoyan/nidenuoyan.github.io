    使用说明

    这是一个金融风险因子生成器，可以根据自然语言描述生成因子定义、数学表达式和 Python 代码。

    注意：需要后端 API 服务

    要使用完整功能，请先启动后端服务：

    • 克隆或下载 Talk Factor Trae 项目
    • 安装依赖：pip install -r requirements.txt
    • 配置 DeepSeek API 密钥（在 config/llm_config.json 中）
    • 启动后端服务：python -m uvicorn backend.main:app --reload --port 8000
    • 访问本页面，并确保后端服务在 http://localhost:8000 运行

    简化版本：如果你只需要因子描述和表达式，可以直接使用自然语言解析功能。

    腾讯云部署方案

    你已经拥有 MySQL、腾讯云、域名和 HTTPS，可以部署完整的动态版本：

    • 后端部署：将 FastAPI 服务部署到腾讯云服务器
    • 数据库：使用现有的 MySQL 存储会话和因子历史
    • 前端配置：修改 static/app.js 中的 API 地址为你的域名
    • HTTPS：配置 SSL 证书确保安全通信
