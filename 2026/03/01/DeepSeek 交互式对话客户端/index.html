DeepSeek 交互式对话代码

功能说明：

  • 支持流式/非流式两种对话模式
  • 自动保存对话历史
  • 支持清空历史、查看历史
  • 使用 DeepSeek API 进行对话
  • 需要配置 .env 文件中的 DEEPSEEK_API_KEY

使用方法：

  1. 安装依赖：pip install openai python-dotenv
  2. 创建 .env 文件，添加 DEEPSEEK_API_KEY=你的密钥
  3. 运行：python interactive_chat.py

项目亮点：

  • 完整的面向对象设计
  • 优雅的错误处理
  • 用户友好的交互界面
  • 支持中英文命令
  • 可扩展的架构设计
